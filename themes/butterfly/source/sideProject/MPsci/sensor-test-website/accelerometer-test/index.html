功能說明

此頁面將測試您的瀏覽器是否可以存取裝置的運動和方向感測器。您將看到即時的加速度計、陀螺儀數據，並可以透過傾斜手機/平板來控制下方畫布中的小球移動。

IPO 範例：

  • I (Input): 手機感測器偵測到運動數據。
  • P (Process): 網頁 JavaScript 處理數據，計算傾斜角度或搖晃強度。
  • O (Output): 網頁上物體移動、旋轉、背景變化。

操作提示： 請先點擊「開始感測器」按鈕。部分瀏覽器（尤其是桌面瀏覽器）可能不支援這些感測器，或者需要您在瀏覽器設定中明確啟用感測器存取。

在手機上，您可能需要在彈出視窗中允許網站存取運動和方向感測器。
