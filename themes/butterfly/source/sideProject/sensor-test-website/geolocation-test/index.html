功能說明

此頁面將測試您的瀏覽器是否可以獲取裝置的地理位置資訊 (GPS)。點擊按鈕後，瀏覽器會請求權限，成功後將顯示您的經緯度、海拔、精確度等資訊。

IPO 範例：

  • I (Input): GPS 模組偵測位置。
  • P (Process): 網頁 JavaScript 處理位置數據，與預設目標位置比較。
  • O (Output): 網頁上顯示距離目標點的遠近、觸發新地圖標記。

操作提示： 請點擊「獲取當前位置」按鈕。瀏覽器將會詢問您是否允許網站存取您的位置資訊。為了獲得較佳的精確度，請確保裝置的 GPS 功能已開啟，且在室外或訊號良好的地方進行測試。
