功能說明

此頁面將測試您的瀏覽器是否可以存取裝置的環境光感測器。如果支援，您將看到目前環境光線的估計亮度值 (通常以 lux 為單位)，下方的指示區域顏色也會隨光線強度變化。

IPO 範例：

  • I (Input): 手機環境光感測器接收光線。
  • P (Process): 網頁 JavaScript 判斷光線閾值。
  • O (Output): 網頁背景顏色變化、虛擬燈泡亮滅。

操作提示： 請先點擊「開始感測器」按鈕。部分瀏覽器可能預設禁用此感測器，您可能需要在瀏覽器設定中啟用「通用感測器額外功能」或類似選項，並且網站必須透過 HTTPS 協定存取。在手機上，您可能需要在彈出視窗中允許網站存取環境光感測器。

如果感測器無法啟動，請檢查您的瀏覽器控制台以獲取更多錯誤訊息。
