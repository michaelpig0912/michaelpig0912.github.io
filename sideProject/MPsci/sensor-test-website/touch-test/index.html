功能說明

此頁面將測試裝置的觸控螢幕功能。您可以在下方的指定區域內用手指（或滑鼠，如果是在桌面瀏覽器上）進行繪圖。同時會顯示觸控點的相關資訊。

IPO 範例：

  • I (Input): 手指觸控螢幕。
  • P (Process): 網頁 JavaScript 判斷觸摸位置、手勢。
  • O (Output): 網頁上繪製線條、觸發動畫。

操作提示： 直接在下方的淺灰色區域內滑動手指或拖曳滑鼠即可繪圖。多點觸控也可能被偵測到（部分效果取決於瀏覽器實作）。
