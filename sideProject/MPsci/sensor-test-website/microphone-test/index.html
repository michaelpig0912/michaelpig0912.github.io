功能說明

此頁面將測試您的瀏覽器是否可以存取麥克風，並即時顯示音頻波形、音量大小和分貝數。還可以偵測拍手等聲音事件。

IPO 範例：

  • I (Input): 手機麥克風接收聲音。
  • P (Process): 網頁 JavaScript 處理聲音數據，分析音量、頻率和分貝數。
  • O (Output): 顯示即時波形圖、分貝數值、音量條和拍手偵測。

操作提示： 點擊「開始麥克風測試」按鈕，瀏覽器會要求麥克風權限。授權後即可看到即時的音頻分析結果。
